You can specify the following types of grids

  • col2 - 50% (2x2)
  • col3 - 33.3% (3x3)
  • col1-3 - 33.3% (33.3% for 3 column)
  • col2-3 - 66% (66.6% for 3 column)
On phones, the grid system will always default to a single row. On tablets, it will display inline.
